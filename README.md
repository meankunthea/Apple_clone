````
# 🍎 Apple Clone

A responsive Apple website clone built with **React**, **Vite**, and **Tailwind CSS**. This project recreates the modern Apple website interface with reusable components and responsive layouts inspired by Apple's official website.

---

## 📖 Project Overview

This project aims to replicate the design and user experience of Apple's website while practicing modern frontend development using React. It focuses on responsive design, reusable UI components, and clean project organization.

---

## ✨ Features

- Responsive Apple-inspired interface
- Reusable React components
- Navigation bar with multiple pages
- Home page
- Store page
- Mac page
- iPad page
- iPhone page
- Product cards
- Hero sections
- Swiper carousel
- Responsive layouts
- Footer

---

## 🛠️ Technologies Used

- React
- Vite
- Tailwind CSS
- React Router DOM
- Swiper
- Lucide React
- JavaScript (ES6+)

## 🚀 Installation

Clone the repository:

```bash
git clone on Gitlab: https://git.clp.kr/anbschool/5th/nextgenit/apple_clone_kunthea.git
git clone on Github: https://github.com/meankunthea/Apple_clone.git
```

Move into the project directory:

```bash
cd Apple_clone
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser:

```text
http://localhost:5173
```

---

## 📦 Production Build

Build the project:

```bash
npm run build

Preview the production build:
````
